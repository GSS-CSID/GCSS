Base

針對 HTML 的元素以及基本文字字級的設定。先定義基本的 h1\~h6，字級可以參考 modulare scale ，注意是”參考”，不是完全依照網頁的規則。

// *************************************////   Base//   -> Tag-level settings//// ************************************* html {  height: 100%;} body {  min-height: 100%;  font-family: "Helvetica Neue", Helvetica, Arial, sans-serif;   font-weight: 300;  ...} // -------------------------------------//   Block Element// ------------------------------------- %h1, h1 {  @include adjust-font-size-to(34px, 2);}%h2, h2 {  @include adjust-font-size-to(27px, 1.5);}%h3, h3 {  @include adjust-font-size-to(21px, 1.25);}%h4, h4 {  @include adjust-font-size-to(16px);}%h5, h5 {  @include adjust-font-size-to(13px);}%h6, h6 {  @include adjust-font-size-to(10px, .875);}p, li {  @include adjust-font-size-to(13px);}     // -------------------------------------//   inline Element// ------------------------------------- a {  @include link-colors($c-link, $c-link-hover);  text-decoration: none;  -webkit-tap-highlight-color: rgba(0, 0, 0, .2);}
